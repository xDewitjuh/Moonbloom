## ✨ Features

- 🌸 Soft pink Moonbloom aesthetic
- 🖼️ Custom wallpaper support
- 🔄 Reset to the default Moonbloom wallpaper
- 🎨 Fully customizable color palette
- 💾 Automatically saves all settings
- ⭐ Animated twinkling stars
- ☄️ Shooting star effects
- 💿 Rotating album artwork while music is playing
- ⚙️ Built-in settings panel

## 🚀 Installation

### Using the Spicetify Marketplace

Search for **Moonbloom** in the Marketplace and install it.

### Manual Installation

1. Download or clone this repository.
2. Copy the **Moonbloom** folder to your Spicetify Themes directory.
3. Apply the theme.

```bash
spicetify config current_theme Moonbloom
spicetify apply
```

---

## 🎨 Customization

Moonbloom includes a built-in settings menu where you can customize:

- Wallpaper
- Theme colors
- Top bar color
- Star colors
- Shooting star colors

All settings are automatically saved and restored when Spotify starts.
